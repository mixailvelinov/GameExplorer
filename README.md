Game Explorer is a web app that allows users to review games and read other players' reviews. 

There are two admin roles: the standard superuser with full CRUD rights, and Review Moderators that can delete inapropraite or ofensive reviews. 

Admins can add games through the admin, directly through the site or via REST API.
